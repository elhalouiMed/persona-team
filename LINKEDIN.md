I got tired of being the bottleneck on my own projects.

As a solo dev, every complex task hits the same wall: you can only hold so much context, work on one thing at a time, and switch roles — analyst, architect, coder, reviewer — sequentially. It's slow and it's exhausting.

So I built Persona Team: a Claude Code add-on that spins up 10 senior AI agents on your existing subscription (no API keys, no per-token billing), decomposes your task, and fans the work out in parallel. Architect designs. Backend and frontend build simultaneously. QA tears it apart. Team-lead integrates and ships.

While all of that happens, you watch a live web dashboard at localhost:7331 — futuristic dark UI, agent cards animating queued → working → done in real time, elapsed timers, activity feed. It feels like standing in a war room.

The part I'm most proud of: the tool built its own launch. I ran `/build-team "ship the Persona Team open-source release"` — BA scoped it, architect planned it, then frontend, devops, and product-owner ran in parallel while I watched the dashboard. QA reviewed. Team-lead integrated. You're reading the output right now.

It's open source, MIT, one-curl install, works in any project.
